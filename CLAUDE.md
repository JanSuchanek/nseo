# nseo

SEO toolkit — meta tags, Open Graph, Twitter Cards, JSON-LD structured data, sitemap.

## Key Classes
- `SeoManager` — set/render meta tags, OG, JSON-LD in Latte
- `SitemapGenerator` — generate XML sitemap
- DI Extension: `SeoExtension`

## Conventions
- PHP 8.2+ with `declare(strict_types=1)`
- PSR-4 autoloading in `src/`
- Nette DI Extension for service registration
- PHPStan level 9
- Part of `jansuchanek/*` on Packagist
- GitHub: https://github.com/JanSuchanek/nseo
